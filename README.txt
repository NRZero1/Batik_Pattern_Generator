Ketika menjalankan program, untuk memuat gambar terdapat pada folder Image
dengan urutan gambar yang harus dimuat sebagai berikut:
1. Load Pattern 1 = Pattern 1.bmp (Pattern bunga)
2. Load Pattern 2 = Pattern 2.bmp (Pattern snowflake)
3. Load Texture = Texture.bmp

File gambar dengan nama Batik Pattern.bmp merupakan hasil motif batik yang dihasilkan
dari program Batik Pattern Generator ini.
